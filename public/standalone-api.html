SlotAI Step Navigation API

This page provides direct access to navigate between steps in the SlotAI application. Use the buttons below to trigger navigation to specific steps.

Direct Step Navigation

Use this endpoint to navigate directly to a specific step.

GET /?step={stepIndex}&force=true

Parameters:

  • step - The zero-based index of the step to navigate to (0 = Step 1, 1 = Step 2, etc.)
  • force - Set to "true" to force navigation regardless of current step
