# Animation Lab Storage

This folder contains saved sprites and animation components for debugging and reuse.

## Structure

- `symbols/` - Original symbol images with metadata
- `sprites/` - Generated sprite sheets with separated components

## Usage

- Images are automatically saved here when generated in the Animation Lab
- Use the "Browse Animation Lab" button to view and reload saved sprites
- Download buttons allow you to save sprites locally for external use

## Debugging

- Each sprite includes metadata about detection results and generation parameters
- Sprite sheets contain separated components for individual animation
- Check console logs for save/load operations
